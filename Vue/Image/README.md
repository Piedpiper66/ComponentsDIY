# Image

| 参数   | 说明            | 类型                                                 | 默认值    |
| ------ | --------------- | ---------------------------------------------------- | --------- |
| src    | 图片地址        | string                                               | undefined |
| fit    | object-fit      | string                                               | cover     |
| size   | 图片的大小/宽高 | string \| number \| [string\|number, string\|number] | undefined |
| avatar | 是否处理为圆形  | boolean                                              | false     |
